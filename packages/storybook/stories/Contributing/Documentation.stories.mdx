# Contributing documentation to Beacon

Documentation for beacon-storybook.braze.com is hosted in several different repositories.

- core beacon documentation is written in storybook format, and hosted at https://github.com/braze-inc/beacon
- styling documentation is written in storybook format, and hosted at https://github.com/braze-inc/beacon-styling
- beacon labs documentation is written in storybook format, and hosted at https://github.com/braze-inc/beacon-labs

Follow the processes outlines in <LinkTo kind="welcome-contributing-contributing-code">Contributing Code</LinkTo> to
open a pull request with documentation changes. Please include a screenshot of storybook with the changes in the pull
request description to expedite approval of the changes.

please reach out on slack before you begin work, DASHI can be reached
on <a href="https://brazetechnology.slack.com/archives/CBDKVUF2P" target="_blank" rel="noopener noreferrer">#beacon-product-system</a>.
