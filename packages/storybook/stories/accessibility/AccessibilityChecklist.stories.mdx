# Checklists & Tools

![a11y logo](/accessibility/a11y_logo.png)
#### [A11y Project Checklist](https://www.a11yproject.com/checklist/)

This checklist uses [The Web Content Accessibility Guidelines (WCAG)](https://www.w3.org/WAI/standards-guidelines/wcag/) as a reference point. The issues this checklist prompts you to check for covers a wide range of disability conditions. It’s important to remember that there is no such thing as “perfect accessibility” or a site being “100% accessible.” <br /> <br /> <br />

![IBM Accessibility Requirements Logo](/accessibility/IBM_Accessibility_Requirements_logo.jpeg)
#### [IBM Accessibility Requirements](https://www.ibm.com/able/requirements/requirements/)
The IBM Accessibility Requirements are a unified list of requirements that incorporate several accessibility standards and regulations. This reference shares what accessibility requirements a product needs to meet. To find out how to meet these requirements, check out [the Equal Access Toolkit](https://www.ibm.com/able/toolkit/)! <br /> <br /> <br />

![Voice Over Screen Reader Logo](/accessibility/Vocie_Over_Screenreader.png)
#### [VoiceOver (Screenreader)](https://webaim.org/articles/voiceover/)
VoiceOver is a screen reader program that comes on new Mac computers, iPhones, iPads, and iPod touches. This article is designed to help users who are new to VoiceOver learn the basic controls for testing web content, and to serve as a reference for the occasional VoiceOver user. <br /> <br /> <br />



## Figma Plugins

![a11y Color Contrast Checker Logo](/accessibility/A11y%20Contrast.png)

#### [a11y - Color Contrast Checker](https://www.figma.com/community/plugin/733159460536249875/A11y---Color-Contrast-Checker)
Ensure there's enough color contrast between text and backgrounds. Build empathy by seeing how others with color blindness see your designs. <br /><br /><br />

![SPELLL - Spell Checking for Figma Logo](/accessibility/Spelll.png)
#### [SPELLL - Spell Checking for Figma](https://www.figma.com/community/plugin/754026612866636376/SPELLL---Spell-Checking-for-Figma)
Ensure designs have the correct spelling so that screen readers can read everything correctly. <br /><br /><br />


![Focus Order Logo](/accessibility/FocusOrderer.png)
#### [Focus Order](https://www.figma.com/community/plugin/731310036968334777/A11y---Focus-Orderer)
Helps designers create designs for keyboarding and screen reader users. <br /><br /><br />


![Focus CSS Logo](/accessibility/Focus%20CSS.png)
#### [Focus CSS](https://www.figma.com/community/plugin/811977401825396882/Focus-CSS)

## Chrome Plugins

![IBM Equal Access Accessibility Checker Logo](/accessibility/IBM_Equal_Access_Checker.jpeg)

#### [IBM Equal Access Accessibility Checker](https://chrome.google.com/webstore/detail/ibm-equal-access-accessib/lkcagbfjnkomcinoddgooolagloogehp)

The IBM Equal Access Accessibility Checker is an open source tool for web developers and auditors that utilizes IBM's accessibility rule engine, which detects accessibility issues for web pages and web applications. The extension integrates into the browser development tools, providing an integrated checking experience, helping users quickly identify the source of accessibility issues and try fixes. <br /><br /><br />


![Spectrum Logo](/accessibility/Spectrum.jpeg)
#### [Spectrum](https://chrome.google.com/webstore/detail/spectrum/ofclemegkcmilinpcimpjkfhjfgmhieb?hl=en)

The Spectrum extension helps you to test web pages for people with different types of CVD. It's particularly useful for websites with data visualizations, because some colors may not be distinguishable from other colors in the charts. <br /><br /><br />


![Headings Map Logo](/accessibility/Headings_Map.jpeg)
#### [HeadingsMap](https://chrome.google.com/webstore/detail/headingsmap/flbjommegcjonpdmenkdiocclhjacmbi?hl=en)

The HeadingsMap extension is useful to show, browse, and audit (for accessibility and SEO) the page’s headings structure. The extension generates a documentmap or index of any web document structured with headings and shows the HTML 5 outline. <br /><br /><br />


![NerdeFocus Logo](/accessibility/NerdeFocus_Logo.jpeg)
#### [NerdeFocus](https://chrome.google.com/webstore/detail/nerdefocus/lpfiljldhgjecfepfljnbjnbjfhennpd)

The NerdeFocus extension displays the focus history with CSS selectors and overlays an easy-to-follow focus indicator on the page. NerdeFocus is great for detecting focus resets, testing keyboard focus, and debugging screen reader issues in modern web apps. <br /><br /><br />
