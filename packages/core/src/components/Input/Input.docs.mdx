# Input

`Input` lets the user input free alphanumeric strings.

### `type` is "number"

Use prop `type === "number"` to make the text field accept only
numeric values. Use props `min` and `max` to set minimum or
maximum values. Use prop `step` to set the increment/decrement
for the number values.

<Canvas>
  <Story id="core-forms-input--type-number" />
</Canvas>

## `type` is "date"

Use prop `type: date` to make the text field accept date
values. This is meant to be a placeholder until we have a proper date
picker.

Use props `min` and `max` to set minimum or
maximum dates. Use prop `step` to set the increment/decrement
for the date values in whole days.

<Canvas>
  <Story id="core-forms-input--type-date" />
</Canvas>

## Properties

| propName    | propType                              | defaultValue | isRequired | Description                                                                           |
| ----------- | ------------------------------------- | ------------ | ---------- | ------------------------------------------------------------------------------------- |
| `className` | string                                | -            | -          | class name of the component                                                           |
| `fullWidth` | boolean                               | -            | -          | takes up 100% of the width of the container                                           |
| `id`        | string                                | -            | -          | id to be passed to the DOM                                                            |
| `invalid`   | boolean                               | -            | -          | indicates that the currently inputted value is invalid                                |
| `max`       | oneOfType( number, string )           | -            | -          | max value for the input field; only relevant when `type` is either `number` or `date` |
| `min`       | oneOfType( number, string )           | -            | -          | min value for the input field; only relevant when `type` is either `number` or `date` |
| `ref`       | `React.RefObject<HTMLInputElement>`   | -            | -          | ref to get access to the DOM node                                                     |
| `step`      | number                                | -            | -          | step to use when bumping up numbers                                                   |
| `type`      | oneOf( `date`, `number` ,`text` )     | -            | -          | type of the input field                                                               |
| `value`     | oneOfType( string, number, string[] ) | -            | -          | value to be rendered in the input field                                               |
| `vSize`     | oneOf( `sm`, `md`)                    | `md`         | -          | vertical size for input box                                                           |
