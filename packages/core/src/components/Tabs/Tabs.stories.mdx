# Tabs

<Meta title="Core/Content Display/Tabs" component={Tabs} />

`Tabs` allow users to see multiple different sections of content grouped by labels.

## Use cases

Below are some common use cases for the `Tabs` component.

### Basic use

`Tabs` are used to create fixed, navigable sections of a page. The `Tab` component
is a navigational label, and the `TabContent` component is content for a specific `Tab`.

### Custom Tab Click use

Each `Tab` can have it's own custom onClick and onKeyPress handler.

> Note: custom key press handlers will need to determine which keys it calls during.
> The `Tab` will call custom key press handlers for ALL key presses.

<Canvas>
  <Story name="custom tab click use">
    {() => {
      const [activeTab, setActiveTab] = React.useState(1);
      const handleSectionThreeClick = () => {
        action("clicked on 3!")();
        setActiveTab(3);
      };
      const handleSectionThreeKeyPress = ({ key, charCode }) => {
        // if 'Space' or 'Enter'
        if (key === " " || charCode === 13) {
          action("keypress on 3!")();
          handleSectionThreeClick();
        }
      };
      return (
        <Box>
          <Tabs onChange={setActiveTab}>
            <Tab tabId={1} active={activeTab === 1}>
              Section 1
            </Tab>
            <Tab tabId={2} active={activeTab === 2}>
              Section 2
            </Tab>
            <Tab
              tabId={3}
              active={activeTab === 3}
              onClick={handleSectionThreeClick}
              onKeyPress={handleSectionThreeKeyPress}
            >
              Section 3
