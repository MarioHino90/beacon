# FormGroup

`FormGroup` organizes form elements together with an optional
legend.

## Use cases

Below are some common use cases for the `FormGroup` component.

### Basic use

`FormGroup` is used to wrap input components to group them together.

<Canvas>
  <Story name="basic use">
    <FormGroup>
      <FormGroupLegend>
        This is a legend: a group of related fields!
      </FormGroupLegend>
      <FieldLabel htmlFor="input-1" label="A form element!">
        <Input id="input-1" />
      </FieldLabel>
      <FieldLabel htmlFor="input-2" label="Another form element!">
        <Input id="input-2" />
      </FieldLabel>
    </FormGroup>
  </Story>
</Canvas>

### No legend

`FormGroup` can also be used without a legend by omitting the prop,
as seen here.

<Canvas>
  <Story name="no legend">
    <FormGroup>
      <FieldLabel htmlFor="input-3" label="A form element!">
        <Input id="input-3" />
      </FieldLabel>
      <FieldLabel htmlFor="input-4" label="Another form element!">
        <Input id="input-4" />
      </FieldLabel>
    </FormGroup>
  </Story>
</Canvas>

### Stacking

Adjacent `FormGroup`s will be stacked with proper spacing and
a horizontal border between them

<Canvas>
  <Story name="stacking">

        <FieldLabel htmlFor="input-6" label="Another form element!">
          <Input id="input-6" />

      </FormGroup>
      <FormGroup>
        <FormGroupLegend>
          This is a legend: another second group of related fields!
        </FormGroupLegend>
        <FieldLabel htmlFor="input-7" label="A form element!">

        </FieldLabel>
        <FieldLabel htmlFor="input-8" label="Another form element!">
          <Input id="input-8" />
        </FieldLabel>
      </FormGroup>
    </React.Fragment>
  </Story>
</Canvas>
