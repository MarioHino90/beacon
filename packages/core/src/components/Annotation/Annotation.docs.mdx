# Annotation

An annotation decorates a block element with an icon indicating a state
or giving the user a hint about an interaction.

Annotations should be used in conjunction with `Tooltip`s or `Popover`s.

### Disabled use

The `Annotation` component can be disabled via the `disabled` prop. This allows
you to only show the annotation when needed without having to duplicate a bunch of 
code.

## Properties

| propName    | propType                            | defaultValue | isRequired | Description                                                          |
| ----------- | ----------------------------------- | ------------ | ---------- | -------------------------------------------------------------------- |
| `children`  | oneOfType( node, arrayOf( node ) )  | -            | -          | children to be passed into component                                 |
| `className` | string                              | -            | -          | class name of the component                                          |
| `disabled`  | boolean                             | -            | -          | indicates that the annotation tooltip and its icon should be visible |
| `id`        | string                              | -            | -          | id to be passed to the DOM                                           |
| `variant`   | oneOf( `info`, `warning`, `error` ) | `info`       | -          | variant that toggles the color and icon of the component             |
