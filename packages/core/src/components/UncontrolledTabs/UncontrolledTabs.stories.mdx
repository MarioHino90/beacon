# UncontrolledTabs

<Meta
  title="Commons/Content Display/UncontrolledTabs"
  component={UncontrolledTabs}
/>

`UncontrolledTabs` allow users to see multiple different sections of content grouped by labels.
It handles the state for the `Tabs` component

## Use cases

Below are some common use cases for the `UncontrolledTabs` component.

### Basic use

`UncontrolledTabs` are used to create fixed, navigable sections of a page.

## Usage

```javascript
import {
  UncontrolledTabs,
  UncontrolledTab,
  UncontrolledTabContent,
  UncontrolledTabsContextProvider,
} from "@braze/beacon-lab-commons-uncontrolled-tabs";
```

## Properties for `UncontrolledTabs`

| propName    | propType                         | defaultValue | isRequired | Description                          |
| ----------- | -------------------------------- | ------------ | ---------- | ------------------------------------ |
| `children`  | oneOf(ReactNode, ReactNodeArray) | -            | -          | children to be passed into component |
| `className` | string                           | -            | -          | class name of the component          |
| `id`        | string                           | -            | -          | id to be passed to the DOM           |

## Properties for `UncontrolledTab`

| propName    | propType                         | defaultValue | isRequired | Description                          |
| ----------- | -------------------------------- | ------------ | ---------- | ------------------------------------ |
| `children`  | oneOf(ReactNode, ReactNodeArray) | -            | -          | children to be passed into component |
| `className` | string                           | -            | -          | class name of the component          |
| `disabled`  | boolean                          | -            | -          | indicates that the tab is disabled   |
| `id`        | string                           | -            | -          | id to be passed to the DOM           |

## Properties for `UncontrolledTabContent`

| propName    | propType                         | defaultValue | isRequired | Description                          |
| ----------- | -------------------------------- | ------------ | ---------- | ------------------------------------ |
| `children`  | oneOf(ReactNode, ReactNodeArray) | -            | -          | children to be passed into component |
| `className` | string                           | -            | -          | class name of the component          |
| `id`        | string                           | -            | -          | id to be passed to the DOM           |

## Properties for `UncontrolledTabsContextProvider`

| propName     | propType                         | defaultValue | isRequired | Description                          |
| ------------ | -------------------------------- | ------------ | ---------- | ------------------------------------ |
| `children`   | oneOf(ReactNode, ReactNodeArray) | -            | -          | children to be passed into component |
| `defaultTab` | oneOf(string, number)            | -            | -          | the default active tab               |
| `onChange`   | (id: number) => void             | -            | -          | callback on active tab change        |
