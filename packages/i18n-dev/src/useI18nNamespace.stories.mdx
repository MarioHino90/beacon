The `useI18nNamespace` hook exported from `"@braze/beacon-translations"` is an alias
to the <code><LinkTo kind="lab-i18n-usei18nnamespace--usage">useI18nNamespace</LinkTo></code>
hook that is typed for usage inside Beacon components.
