# AlertModal

The `AlertModal` is a commons component that aims to abstract away
the frequent need to alert the user that something did or will happen
based on an action they may take.

## Use cases

Below are some common use cases for the `AlertModal` component.

### Basic use

To indicate the type of information that is being provided, you can set
`intent`. By default, `intent` is set to `info` as shown below.

> Tip: Head over to the `Canvas` tab to access the various knobs
> available for this component.

## Usage

```javascript
import { AlertModal, useAlertModal } from "@braze/beacon-commons";
```

## Properties

| propName        | propType                                                | defaultValue | isRequired | Description                                                                            |
| --------------- | ------------------------------------------------------- | ------------ | ---------- | -------------------------------------------------------------------------------------- |
| `buttonText`    | string                                                  | -            | +          | text for button inside modal footer                                                    |
| `children`      | oneOfType( node, arrayOf( node ) )                      | -            | -          | children to be passed into component                                                   |
| `className`     | string                                                  | -            | -          | class name of the component                                                            |
| `heading`       | string                                                  | -            | +          | modal heading text                                                                     |
| `id`            | string                                                  | -            | -          | id to be passed to the DOM                                                             |
| `intent`        | oneOf( `info`, `danger`, `warning`, `success` )         | `info`       | -          | intent of the alert modal representing the type of information that should be conveyed |
| `onButtonClick` | `(event: React.MouseEvent<HTMLButtonElement>) => void;` | -            | +          | function to be called when user clicks on button in modal footer                       |
| `size`          | oneOf(`sm`, `md`, `lg`, `xl`)                           | `md`         | -          | size of the `Modal` to be displayed                                                    |
